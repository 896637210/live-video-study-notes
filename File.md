### 构造函数
File()：返回一个新构建的文件对象（File）

### 实例属性
File 接口也继承了 Blob 接口的属性

file.lastModified：返回当前 File 对象所引用文件最后修改时间， 自 1970年1月1日0:00 以来的毫秒数

file.name：返回当前 File 对象所引用文件的名字

file.size：返回文件的大小

file.type：返回文件的扩展类型

### 实例方法
File 接口没有定义任何方法，但是继承了 Blob 接口的方法
